# 🏗️ Laravel Construction Management System

## 📌 Overview
This project is a full-stack construction management system developed using Laravel. It allows users to manage services, projects, and content through an admin dashboard.

## 🚀 Features
- Admin panel for managing services
- Image upload and storage system
- Dynamic service listing (React frontend)
- REST API integration
- Authentication system

## 🛠️ Technologies Used
- Laravel (Backend)
- React (Frontend)
- MySQL (Database)
- REST APIs

## 📂 Project Structure
- `/app` → Laravel backend logic
- `/resources` → frontend views
- `/public/uploads` → images
